# Food-Ordering-System
Food Ordering System – Python CLI Application  A simple and interactive command-line based Food Ordering System developed using Python. Users can select items from a predefined menu, review their order, and modify it before finalizing. The project demonstrates practical use of loops, conditional statements, list operations, and user input handling.
This is a Python-based command-line food ordering system.
Users can place an order by selecting items from a menu and later modify the order by adding, deleting, or updating items.

Features

✔ Displays menu options
✔ Add multiple items to an order
✔ Modify order (Add / Delete / Replace item)
✔ Validates choices
✔ Final bill preview

Tech Used

Python Lists

Conditional Statements

Loops

User Input Handling
